# Ecommerce de tecnologia

## Sobre este proyecto
Este proyecto se realiza para el curso de REACT de Coderhouse.
La idea de diseño es basada en los colores y fuentes de la película Matrix, las funcionalidades fueron realizadas para satisfacer las consignas de la entrega final.
El proyecto fue construido con REACT, VITE, JS y BOOTSTRAP.

Se usó la librería Toastify para mostrar notificaciones de forma simple.
